C Programming Syllabus

Module 1: Introduction to Programming Concepts

  • History and Importance of C
  • Structure of a C program
  • Compilation and Execution
  • IDEs and Basic Setup (GCC, Code::Blocks)

Module 2: Variables, Data Types and Operators

  • Keywords, Identifiers, Constants
  • Data types: int, float, char, double
  • Arithmetic, Logical, Relational, Bitwise operators
  • Type conversion and typecasting

Module 3: Control Flow and Loops

  • Conditional Statements: if, if-else, nested if
  • switch statement
  • Loops: for, while, do-while
  • break, continue, and goto

Module 4: Functions in C

  • Function declaration and definition
  • Argument passing (call by value and reference)
  • Recursion
  • Scope and lifetime of variables

Module 5: Arrays and Strings

  • 1D and 2D Arrays
  • String manipulation and library functions
  • Array of strings

Module 6: Pointers and Dynamic Memory Allocation

  • Pointer basics and operations
  • Pointer arithmetic
  • Pointers and arrays
  • malloc(), calloc(), realloc(), free()

Module 7: Structures and Unions

  • Defining and using structures
  • Nested structures
  • Unions
  • Structures with functions and pointers

Module 8: File Handling in C

  • File operations: open, read, write, close
  • Modes: r, w, a, rb, wb
  • File pointers and error handling

Module 9: Preprocessor Directives and Header Files

  • #define, #include, #ifdef, #ifndef
  • Creating and including user-defined header files

Module 10: Mini Projects and Assignments

  • Calculator, Student Record System, File Encryptor
  • Weekly assignments and lab exercises

Evaluation Criteria

  • Quizzes and assignments: 30%
  • Mid-term practical test: 20%
  • Final project: 25%
  • Viva and written test: 25%
